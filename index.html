    MKDocs介绍¶

    MKDocs是可以集成python的markdown插件，有很强的表现力。

    安装¶

    • 使用pip安装
    1
2
3
4
5
    $ pip install mkdocs

$ mkdocs new <your-project>
$ cd <your-project>
$ mkdocs serve
    说明

    如果pip速度太慢，使用 pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple进行加速

    • 安装相关插件
    1
2
3
4
5
6
7
8
    # 安装主题
pip install mkdocs-material

# 安装代码高亮
pip install pyments

# 安装插件
pip install pymdown-extensions

  name: 'material'

      smart_enable: all

  - pymdownx.details
  - pymdownx.emoji:
      emoji_generator: !!python/name:pymdownx.emoji.to_svg
  - pymdownx.inlinehilite
  - pymdownx.magiclink
  - pymdownx.mark
  - pymdownx.smartsymbols
  - pymdownx.superfences
  - pymdownx.tasklist:
      custom_checkbox: true
  - pymdownx.tilde

extra_javascript:
  - 'https://cdnjs.cloudflare.com/ajax/libs/mathjax/2.7.0/MathJax.js?config=TeX-MML-AM_CHTML'

    Extensions¶

    这里列出比较常用的插件

    • Critic: 提供更多的行内样式, 如下

      • insert me
      • remove me
      • hightlight me

    • arithmatex: 支持$和$$等数学表达

    • superfences: 支持admontion等样式
    • mark: 高亮1
