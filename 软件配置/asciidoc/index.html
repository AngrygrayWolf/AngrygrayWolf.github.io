    asciidoctor¶

    asccidoctor比markdown有更加强的表现力。我一般大部分写作使用markdown，论文和项目相关的文章采用asciidoc。

    使用¶

    基础语法¶

    asciidoc支持部分markdown语法，但是仍然建议使用 asciidoc 自带的语法。

    • heading使用=

    • admotion block 使用 [IMPORTANT]， 如：

    1
    this is a test
    • 代码：
